# サイト基本設計

## サイト種別

1ページ完結型の静的ポートフォリオサイト。

## ページ構成

1. Hero
   - 氏名
   - 肩書き
   - キャッチコピー
   - 主要対応領域

2. Profile
   - 経歴要約
   - AI/Web領域への再学習

3. Strengths
   - 幅広い技術領域
   - 上流から下流までの一貫対応
   - レガシーと現代技術の橋渡し
   - 業務システム・基幹系理解
   - AI活用型開発への転換

4. Skills
   - Languages
   - Database
   - OS / Environment
   - Process
   - Business Domains

5. Projects
   - 代表プロジェクトをカード形式で表示
   - 各カードは概要、担当、技術、訴求点を持つ

6. Career
   - 技術領域の変遷
   - 長期実務経験と現在の学習状況

7. AI / Web Learning
   - AI×Webクリエイター養成科
   - AIツール活用方針
   - 今後追加する成果物

8. Contact
   - GitHub等の公開導線

## デザイン方針

- 落ち着いた配色
- 高い可読性
- 実務経験者としての信頼感
- 派手なアニメーションより情報整理を優先
- スキルタグとプロジェクトカードで視認性を高める
