# 作業エラー履歴

このファイルは、portfolioリポジトリで利用する軽量エラー履歴の初期テンプレートです。

## 記録方針

- 詳細ルールは `chatgpt-ops-rules` を正本とする。
- APIM、Q-Scout、その他リポジトリ固有の履歴をここへコピーしない。
- 公開ポートフォリオの安全性方針を優先する。
- 個人情報、実企業名、顧客名、内部システム名、秘密情報、未公開の外部情報を記録しない。
- 通常TSKへ昇格すべき内容は、portfolio側の改善タスク管理または `chatgpt-ops-rules` の分離方針に従って個票化する。

## エントリ

現在、portfolio固有の軽量エラー履歴はありません。
